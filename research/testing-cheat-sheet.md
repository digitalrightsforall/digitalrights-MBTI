# 数字灵魂 MBTI (DCP 2.0) 测试路径指南

本指南用于协助测试 16 种人格的生成逻辑。

## 1. 核心计分逻辑速查
- **1-6 题**: A = 主权(S), B = 效用(U)
- **7-12 题**: A = 审慎(C), B = 乐观(O)
- **13-18 题**: A = 掌控(M), B = 让渡(D)
- **19-24 题**: A = 积极(P), B = 响应(R)

## 2. 典型人格达成路径 (全选示例)

| 目标人格 | 1-6 题 | 7-12 题 | 13-18 题 | 19-24 题 | 备注 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **SUMP (主权捍卫者)** | 全选 A | 全选 A | 全选 A | 全选 A | 最硬核的防御者 |
| **UODR (全自动咸鱼)** | 全选 B | 全选 B | 全选 B | 全选 B | 最彻底的便利追求者 |
| **UODP (AI 驯化大师)** | 全选 B | 全选 B | 全选 B | 全选 A | 追求效率且懂配置 |
| **SCMP (数据洁癖狂)** | 全选 A | 全选 A | 全选 A | 全选 A | 与 SUMP 类似，侧重算法怀疑 |
| **UOMR (社畜生存狂)** | 全选 B | 全选 B | 全选 A | 全选 B | 享受效用但行为受控且疲劳 |
| **SCDP (维权战神)** | 全选 A | 全选 A | 全选 B | 全选 A | 怀疑算法但敢于让 Agent 试错以抓把柄 |

## 3. 极简测试建议
如果您想快速刷出某个特定结果，可以记住这四个“六题区间”：
1. **[1-6]** 选 A 为 S，选 B 为 U
2. **[7-12]** 选 A 为 C，选 B 为 O
3. **[13-18]** 选 A 为 M，选 B 为 D
4. **[19-24]** 选 A 为 P，选 B 为 R

## 4. 特殊权重提醒
以下题目分值为 **1.5 (Critical)**，对结果影响更大：
- 第 2 题 (政务免填表) -> S/U 关键
- 第 8 题 (招聘算法) -> C/O 关键
- 第 13、15、17 题 (钱包、社交、一键授权红线) -> M/D 关键
- 第 20、24 题 (杀熟投诉、日志上传) -> P/R 关键
