# 数字灵魂 MBTI (Digital Rights Personality Assessment)

> 2026 AI 时代：你是驯服算法的大师，还是被看光的透明人？

本项项目是一个面向普通市民的公益评测系统，旨在提升全民科技伦理素养，增强数字权利意识。通过 24 道基于真实中国职场与社交场景的情境测试，深度解析用户的数字主权、算法信任、代理授权与能动性。

## 🌟 项目愿景
「普通人的数字权利」是一个面向普通市民开展提升全民科技伦理素养，提升全民数字权利意识，为普通人提供数字权益发声和参与平台的公益社区。

## 🧠 理论框架 (DCP 2.0)
本项目基于深度调研报告，确立了四个核心认知维度：
- **S / U (数据主权 vs. 极致效用)**: 隐私微积分与 PIPL 权利博弈。
- **C / O (算法审慎 vs. 技术乐观)**: 算法黑盒焦虑与全球信任赤字。
- **M / D (自主掌控 vs. 代理让渡)**: Agentic AI 授权红线与认知延展边界。
- **P / R (积极维护 vs. 被动响应)**: 权利觉醒行动与隐私疲劳。

## 🎨 16 种数字公民人格
从极度敏锐的「维权战神」到极致便利的「全自动咸鱼」，本项目定义了 16 种极具社会共鸣的人格画像。

## 🛠 技术栈
- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS & Vanilla CSS (Vibrant Tech Theme)
- **Animation**: Framer Motion
- **Visualization**: Recharts (Radar Charts)
- **Icons**: Lucide React

## 📦 安装与部署

### 开发环境
```bash
cd src/web-app
npm install
npm run dev
```

### 部署到 GitHub Pages
项目已配置自动化静态导出。运行以下命令即可部署：
```bash
npm run deploy
```
